# Ingeniería de Software II - TP1
Este repositorio contiene el Trabajo Práctico N°1 para la cátedra de Ingeniería de Software II de la FCyT.

## Propósito del Repositorio
El objetivo de este proyecto es demostrar el manejo de Git, GitHub y la aplicación de buenas prácticas en el desarrollo de software.

### Requisitos del Trabajo
Para completar esta entrega se deben cumplir los siguientes puntos:
1. Configuración del entorno local.
2. Gestión de ramas (branches) en Git.
3. Documentación clara mediante Markdown.
